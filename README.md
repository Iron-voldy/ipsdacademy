## ğŸ¨ Customization

### Colors
The website uses CSS custom properties for easy color customization. Edit the `:root` section in `css/style.css`:

```css
:root {
    --primary-600: #0095de;    /* Main brand color */
    --secondary-600: #f48422;  /* Accent color */
    /* ... other colors */
}
```

### Content
- Update text content in `index.html`
- Replace images in the `assets/` folder
- Modify contact information and social links

### Styling
- All styles are in `css/style.css`
- Responsive breakpoints are clearly defined
- Component-based CSS organization
